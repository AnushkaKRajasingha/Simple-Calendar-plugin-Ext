== Changelog ==

= 1.0.8 - January 21, 2019 =

* Feature: Added additional time formatting setting for time displayed with the event title.

= 1.0.7 - January 9, 2019 =

* Fix: Issue with language files not working.

= 1.0.6 - August 30, 2018 =

* Fix: Issue with Appearance > Display Language menu would be empty.
* Fix: Event title cutoff due to nowrap styling. Makes this consistent with base plugin.

= 1.0.5 - June 11, 2018 =

* Fix: Issue where an event that starts non-DST but ends in DST would display for an additional day.

= 1.0.4 - September 12, 2016 =

* Feature: Set default start time for week and day views.
* Feature: Limit display times for week and day views.
* Feature: Select a language to display visual elements in.
* Fix: Fix single quotes rendered as HTML codes.

= 1.0.3 - September 1, 2016 =

* Fix: Prevent non-all day events extending to all day in week view (introduced in 1.0.2).

= 1.0.2 - August 25, 2016 =

* Fix: All day multi-day events will now show their full length.
* Fix: Make qTip styles in add-on more consistent with those in core plugin.
* Dev: Removed unnecessary files from build.
* Dev: Tested up to WordPress 4.6.

= 1.0.1 - May 30, 2016 =

* Fix: Default view setting now saves and displays week or day view properly.

= 1.0.0 - May 25, 2016 =

* Initial Release.
